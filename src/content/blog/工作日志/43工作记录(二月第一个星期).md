## ollama是什么？

**Ollama** 是一个开源的 AI 对话平台，它致力于提供一个高效、可自定义、易于部署的人工智能对话系统。Ollama 允许开发者构建、训练和部署基于大规模预训练模型的对话系统（类似于GPT系列模型、BERT等）。这个平台旨在帮助用户和开发者更轻松地将先进的语言模型集成到他们的应用中。

### Ollama的关键特点：

1. **开源和易用性**：
   - Ollama 是开源的，意味着任何人都可以查看源代码、修改、贡献以及根据需求进行自定义。它的目标是简化构建和训练大规模语言模型的过程。
2. **多模型支持**：
   - Ollama 支持多种预训练模型，包括类似 GPT（生成预训练变换器）以及其他对话生成模型。这些模型可用于各种任务，如对话生成、文本摘要、情感分析等。
3. **可自定义和灵活**：
   - 开发者可以根据具体需求调整模型配置，例如调整层数、隐藏层大小、优化器选择等，以满足特定应用的需求。
4. **部署和推理**：
   - Ollama 使得模型部署变得更加简便，并优化了推理性能。这使得它可以在各种硬件环境中进行高效运行。
5. **API支持**：
   - Ollama 提供API支持，开发者可以通过简单的接口调用与模型进行交互，进行文本生成、语义理解等操作。

### 典型应用场景：

- **对话系统**：为聊天机器人或客服系统提供支持，能够进行自然语言对话。
- **文本生成**：基于给定的上下文生成连贯且有创意的文本内容。
- **语义理解**：分析文本的含义，执行情感分析、主题建模等任务。
- **自动化任务**：集成到自动化工作流中，如自动生成报告、邮件响应等。

Ollama 的目标是使AI对话和文本生成更加普及，帮助开发者快速构建与部署高效且灵活的语言模型。****

## 1.下载 Ollama 可执行文件

```bash
curl -L https://ollama.com/download/ollama-linux-arm64.tgz -o ollama-linux-arm64.tgz
sudo tar -C /usr -xzf ollama-linux-arm64.tgz
```

## 2.配置 Ollama

在 /etc/systemd/system/ 下创建服务文件 ollama.service：

```bash
sudo vi /etc/systemd/system/ollama.service
```

在 vi 界面，输入 **i** 进入输入模式，然后添加下面内容：

```bash
[Unit]
Description=Ollama Service
After=network-online.target

[Service]
ExecStart=/usr/bin/ollama serve
User=root
Group=root
Restart=always
RestartSec=3

[Install]
WantedBy=default.target
```

然后按 ESC 键位，输入 `：wq`保存即可。

接下来配置 Ollama，自定义模型存放路径：

```bash
sudo systemctl edit ollama.service
```

同样用上面的方法，将下面的内容输入进去。**在这里/gewu/这个文件夹就是我指定的模型存放路径，大家可以根据自己需求进行更改，比如/home/geduer也可以。**

```bash
[Service]
Environment="OLLAMA_MODELS=/gewu/"
```

## 3.启动 Ollama 服务

在输入下面命令即可启动：

```bash
sudo systemctl daemon-reload
sudo systemctl enable ollama
sudo systemctl start ollama
```

这三条命令只需要输入一次，即使关机下次重新开机，也不需要再输入了。

## 4.使用 Ollama

考虑到性能有限，在这里我们运行的大语言模型为1.5b，在命令行输入下面命令即可：

```bash
ollama run deepseek-r1:1.5b
ollama run deepseek-r1:7b
```

~~~shell
#!/bin/bash

# 设置刷新间隔（秒）
REFRESH_INTERVAL=1

# 函数：计算CPU使用率
get_cpu_usage() {
    # 读取/proc/stat的第一行（总CPU使用情况）
    CPU=($(grep '^cpu ' /proc/stat))
    
    # 计算总CPU时间
    CPU_TOTAL_TIME=0
    for i in "${CPU[@]:1:8}"; do
        CPU_TOTAL_TIME=$((CPU_TOTAL_TIME + i))
    done
    
    # 计算CPU空闲时间
    CPU_IDLE_TIME=${CPU[4]}
    
    # 计算CPU使用时间
    CPU_USAGE_TIME=$((CPU_TOTAL_TIME - CPU_IDLE_TIME))
    
    # 返回CPU使用率和总时间
    echo "$CPU_USAGE_TIME $CPU_TOTAL_TIME"
}

# 初始化上一次的CPU使用时间
LAST_CPU_USAGE=($(get_cpu_usage))
LAST_CPU_USAGE_TIME=${LAST_CPU_USAGE[0]}
LAST_CPU_TOTAL_TIME=${LAST_CPU_USAGE[1]}

# 清屏并显示标题
clear
echo

# 无限循环，实时刷新NPU负载和CPU使用率
while true; do
    # 读取NPU负载信息
    LOAD_INFO=$(cat /sys/kernel/debug/rknpu/load)
    
    # 获取当前CPU使用情况
    CURRENT_CPU_USAGE=($(get_cpu_usage))
    CURRENT_CPU_USAGE_TIME=${CURRENT_CPU_USAGE[0]}
    CURRENT_CPU_TOTAL_TIME=${CURRENT_CPU_USAGE[1]}
    
    # 计算CPU使用率
    CPU_USAGE_DIFF=$((CURRENT_CPU_USAGE_TIME - LAST_CPU_USAGE_TIME))
    CPU_TOTAL_DIFF=$((CURRENT_CPU_TOTAL_TIME - LAST_CPU_TOTAL_TIME))
    CPU_USAGE_PERCENT=$((100 * CPU_USAGE_DIFF / CPU_TOTAL_DIFF))
    
    # 更新上一次的CPU使用时间
    LAST_CPU_USAGE_TIME=$CURRENT_CPU_USAGE_TIME
    LAST_CPU_TOTAL_TIME=$CURRENT_CPU_TOTAL_TIME
    
    # 清屏并显示当前时间
    clear
    echo
    echo "Current Time: $(date '+%Y-%m-%d %H:%M:%S')"
    echo
    
    # 显示CPU使用率
    echo "CPU Usage: ${CPU_USAGE_PERCENT}%"
    echo
    
    # 显示NPU负载信息
    echo "$LOAD_INFO"
    
    # 等待指定的刷新间隔
    sleep $REFRESH_INTERVAL
done
~~~
