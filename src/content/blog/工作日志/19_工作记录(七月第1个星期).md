假如我们创建一个 1080x1920 (高x宽)的视频，每个像素占用 3 bytes 对颜色进行编码(或使用 24 bit 真色彩, 这可以提供 16,777,216 种不同的颜色)，每秒 24 帧，视频时长为 30 分钟。

1. 每帧的大小计算:
   - 视频分辨率为 1080 x 1920
   - 每个像素占用 3 bytes (24 bit 真彩色)
   - 每帧的总像素数为 1080 x 1920 = 2,073,600 个
   - 每帧的大小为 2,073,600 个像素 x 3 bytes/像素 = 6,220,800 bytes
2. 总帧数计算:
   - 视频时长为 30 分钟 = 1,800 秒
   - 视频帧率为 24 fps
   - 总帧数为 1,800 秒 x 24 fps = 43,200 帧
3. 总数据量计算:
   - 每帧 6,220,800 bytes
   - 总帧数 43,200 帧
   - 总数据量为 6,220,800 bytes/帧 x 43,200 帧 = 268,916,736,000 bytes ≈ 250.28 GB
4. 带宽计算:
   - 总数据量为 268,916,736,000 bytes
   - 视频时长为 1,800 秒
   - 带宽 = 总数据量 / 时长 = 268,916,736,000 bytes / 1,800 seconds ≈ 149,398,186 bytes/s
   - 换算成比特每秒(bps)为 149,398,186 bytes/s x 8 bits/byte = 1,195,185,488 bps ≈ 1.19 Gbps

~~~shell
parseargs()

    if [ "x$#" == "x0" ]; then
        return 0
    fi

    while [ "x$#" != "x0" ];
    do
        if [ "x$1" == "x-h" -o "x$1" == "x--help" ]; then
            return 1
        elif [ "x$1" == "x" ]; then
            shift
        elif [ "x$1" == "x-n" -o "x$1" == "x--name" ]; then
            name=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-k" -o "x$1" == "x--kernel" ]; then
            kernel_url=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-b" -o "x$1" == "x--branch" ]; then
            branch=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-c" -o "x$1" == "x--config" ]; then
            config=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-r" -o "x$1" == "x--repo" ]; then
            repo_file=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-d" -o "x$1" == "x--device-tree" ]; then
            dtb_name=`echo $2`
            shift
            shift
        elif [ "x$1" == "x-s" -o "x$1" == "x--spec" ]; then
            spec_param=`echo $2`
            shift
            shift
        else
            echo `date` - ERROR, UNKNOWN params "$@"
            return 2
        fi
    done
}
~~~

参数判断：

1. 如果没有传入任何参数,函数直接返回 0。
2. 循环检查每个参数:

   - 如果参数为空,则跳过。
   - 如果参数是 `-n` 或 `--name`，则将第二个参数的值赋给变量 `name`。
   - 如果参数是 `-k` 或 `--kernel`，则将第二个参数的值赋给变量 `kernel_url`。
   - 如果参数是 `-b` 或 `--branch`，则将第二个参数的值赋给变量 `branch`。
   - 如果参数是 `-c` 或 `--config`，则将第二个参数的值赋给变量 `config`。
   - 如果参数是 `-r` 或 `--repo`，则将第二个参数的值赋给变量 `repo_file`。
   - 如果参数是 `-d` 或 `--device-tree`，则将第二个参数的值赋给变量 `dtb_name`。
   - 如果参数是 `-s` 或 `--spec`，则将第二个参数的值赋给变量 `spec_param`。

### 构建rootfs

~~~shell
build_rootfs() {
    # 在函数退出时调用 UMOUNT_ALL 进行清理
    trap 'UMOUNT_ALL' EXIT
    cd $workdir

    # 如果 rootfs 目录存在，则删除
    if [ -d rootfs ]; then rm -rf rootfs; fi
    mkdir rootfs

    # 如果临时目录不存在，则创建；否则清空
    if [ ! -d ${tmp_dir} ]; then
        mkdir -p ${tmp_dir}
    else
        rm -rf ${tmp_dir}/*
    fi

    # 根据 spec_param 的值进行配置
    if [ "x$spec_param" == "xheadless" ] || [ "x$spec_param" == "x" ]; then
        :
    elif [ "x$spec_param" == "xxfce" ] || [ "x$spec_param" == "xukui" ] || [ "x$spec_param" == "xdde" ]; then
        CONFIG_RPM_LIST=$workdir/../configs/rpmlist-${spec_param}
    elif [ -f ${spec_param} ]; then
        cp ${spec_param} ${tmp_dir}/
        spec_file_name=${spec_param##*/}
        CONFIG_RPM_LIST=${tmp_dir}/${spec_file_name}
    else
        echo `date` - ERROR, please check your params in option -s or --spec.
        exit 2
    fi

    # 初始化 RPM 数据库
    mkdir -p ${rootfs_dir}/var/lib/rpm
    rpm --root  ${rootfs_dir}/ --initdb

    # 检查 repo_file 参数
    if [ "x$repo_file" == "x" ] ; then
        echo `date` - ERROR, \"-r REPO_INFO or --repo REPO_INFO\" missing.
        help 2
    elif [ "x${repo_file:0:4}" == "xhttp" ]; then
        # 处理 http URL 的 repo_file
        if [ "x${repo_file:0-5}" == "x.repo" ]; then
            repo_url=${repo_file}
            wget ${repo_file} -P ${tmp_dir}/
            repo_file_name=${repo_file##*/}
            repo_file=${tmp_dir}/${repo_file_name}
        else
            repo_file_name=tmp.repo
            repo_file_tmp=${tmp_dir}/${repo_file_name}
            index=1
            for baseurl in ${repo_file// / }
            do
                echo [repo${index}] >> ${repo_file_tmp}
                echo name=repo${index} to build rk3399 image >> ${repo_file_tmp}
                echo baseurl=${baseurl} >> ${repo_file_tmp}
                echo enabled=1 >> ${repo_file_tmp}
                echo gpgcheck=0 >> ${repo_file_tmp}
                echo >> ${repo_file_tmp}
                index=$(($index+1))
            done
            repo_file=${repo_file_tmp}
        fi
    else
        # 处理本地文件的 repo_file
        if [ ! -f $repo_file ]; then
            echo `date` - ERROR, repo file $repo_file can not be found.
            exit 2
        else
            cp $repo_file ${tmp_dir}/
            repo_file_name=${repo_file##*/}
~~~
