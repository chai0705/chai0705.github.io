
| 参数               | 全称 / 含义                  | 你的配置解释                                                 |
| ------------------ | ---------------------------- | ------------------------------------------------------------ |
| `-s 86400`         | `seconds` 测试持续时间（秒） | **连续测试 24 小时**（86400 秒 = 1 天）                      |
| `-i 4`             | `IO threads` IO 读写线程数   | 开启**4 个 IO 线程**，压测磁盘 / 存储读写                    |
| `-C 4`             | `CPU threads` CPU 线程数     | 开启**4 个 CPU 线程**，满载压测 4 核 CPU                     |
| `-W`               | `Wide copy` 内存宽拷贝测试   | **严苛内存压测**（比普通测试更严格，专门查内存故障）         |
| `--stop_on_errors` | 检测到错误立即停止           | 只要 CPU / 内存 / IO 出任何报错，**立刻终止测试**（方便定位故障） |
| `-M 300`           | `Memory` 测试内存大小（MB）  | 分配**300MB 内存**用于压力测试                               |
| `-l /路径`         | `log` 日志文件路径           | 把所有测试过程、报错、结果，保存到这个日志文件里             |
