16|# 结构体
17|
18|​	
19|
20|`struct`是C语言中的关键字，用于定义自定义的数据类型，可以将多个不同类型的变量组合在一起形成一个结构体，从而创建一个包含多个成员的复合数据类型。
21|
22|结构体的定义通常包括以下几个部分：
23|
24|1. `struct`关键字：用于声明一个结构体类型。
25|2. 结构体标签：用于标识结构体类型的名称，可以在定义时指定，也可以省略。
26|3. 成员列表：定义结构体的成员，每个成员包括类型和名称。
27|
28|下面是一个简单的示例，展示了如何使用`struct`定义一个包含姓名和年龄的人员信息结构体：
29|
30|```c
31|struct Person {
32|    char name[50];
33|    int age;
34|};
35|```
36|
37|在上面的示例中，`struct`关键字用于定义一个名为`Person`的结构体类型。该结构体有两个成员：`name`和`age`，分别表示人员的姓名和年龄。
38|
39|定义结构体后，就可以声明结构体变量并使用它们，如下所示：
40|
41|```c
42|struct Person person1;
43|struct Person person2;
44|
45|strcpy(person1.name, "John Smith");
46|person1.age = 30;
47|
48|printf("Person 1: Name = %s, Age = %d\n", person1.name, person1.age);
49|
50|person2 = person1;  // 结构体变量之间可以进行赋值操作
51|
52|printf("Person 2: Name = %s, Age = %d\n", person2.name, person2.age);
53|```
54|
55|在上面的示例中，我们声明了两个`Person`类型的结构体变量`person1`和`person2`。然后，我们使用`strcpy`函数将字符串"John Smith"复制到`person1`的`name`成员中，并将30赋值给`person1`的`age`成员。最后，我们打印了`person1`和`person2`的姓名和年龄。
56|
57|结构体变量的成员可以通过`.`运算符来访问，即`结构体变量名.成员名`。可以像访问普通变量一样访问结构体变量的成员。
58|
59|使用结构体可以更好地组织和管理相关数据，将多个相关的数据打包在一起，方便进行传递和处理。结构体还可以嵌套定义，允许在一个结构体中包含另一个结构体作为成员，从而形成更复杂的数据结构。
60|
61|示例代码1：
62|~~~c
63|#include <stdio.h>
64|
65|struct Person {
66|    char *name;
67|    int age;
68|};
69|    
70|int main()
71|{
72|    struct Person person1;
73|    struct Person person2;
74|
75|    person1.name = "John Smith";
76|    person1.age = 30;
77|
78|    printf("Person 1: Name = %s, Age = %d\n", person1.name, person1.age);
79|    
80|    person2 = person1;  // 结构体变量之间可以进行赋值操作
81|    
82|    printf("Person 2: Name = %s, Age = %d\n", person2.name, person2.age);
83|}
84|~~~

217|# typedef 别名
218|
219|`typedef` 是 C 语言中的一个关键字，用于为已有的数据类型创建新的别名。通过 `typedef`，我们可以为现有的数据类型赋予一个新的名称，以便在程序中更方便地使用。
220|
221|`typedef` 的语法如下：
222|
223|```c
224|typedef <existing_data_type> <new_alias>;
225|```
226|
227|其中，`existing_data_type` 是现有的数据类型，可以是基本的数据类型（如 `int`、`float`、`char` 等）或自定义的数据类型（如结构体、枚举等），`new_alias` 是我们为该数据类型创建的新的别名。
228|
229|使用 `typedef` 的主要优点是提高代码的可读性和可维护性。通过为数据类型创建别名，我们可以在代码中使用更具有描述性的名称，从而使代码更易于理解。
230|
231|以下是一些 `typedef` 的示例：
232|
233|```c
234|typedef int Integer;
235|```
236|
237|这个例子中，我们为 `int` 类型创建了一个新的别名 `Integer`。现在，我们可以使用 `Integer` 来定义整型变量，就像使用 `int` 一样：
238|
239|```c
240|Integer num = 10;
241|```
242|
243|另一个常见的应用是在定义结构体时使用 `typedef`，以便在创建结构体变量时不需要每次都加上 `struct` 关键字。例如：
244|
245|```c
246|typedef struct {
247|    char *name;
248|    int age;
249|} Person;
250|```
251|
252|在这个例子中，我们为匿名的结构体类型创建了别名 `Person`。现在，我们可以直接使用 `Person` 来定义结构体变量，而不需要每次都写 `struct` 关键字：
253|
254|```c
255|Person person1;
256|```
257|
258|`typedef` 还可以用于创建函数指针类型的别名，以简化复杂的函数指针声明。例如：
259|
260|```c
261|typedef int (*MathOperation)(int, int);
262|```
263|
264|在这个例子中，我们为接受两个整数参数并返回一个整数的函数指针类型创建了别名 `MathOperation`。现在，我们可以使用 `MathOperation` 来声明函数指针变量，而不需要每次都写出完整的函数指针声明：
265|
266|```c
267|MathOperation add;
268|```
269|
270|总结起来，`typedef` 允许我们为已有的数据类型创建新的别名，提高代码的可读性和可维护性。通过使用 `typedef`，我们可以在代码中使用更具有描述性的名称，使代码更易于理解和维护。
