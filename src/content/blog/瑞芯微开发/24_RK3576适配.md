61|ddrbin_tool工具简介
62|
63|`ddrbin_tool` 是一款用于处理 DDR 配置文件（ddr.bin）的工具。它提供了两种功能，分别用于修改 DDR 配置以及从 DDR 配置文件中提取配置参数。
64|
65|**1.修改 `ddr.bin` 文件（功能1）**
66|
67|该功能通过读取 `ddrbin_param.txt` 文件中的配置参数，生成新的 `ddr.bin` 文件。可以修改 DDR 频率、UART 信息等内容。
68|
69|**步骤**：
70|
71|修改 `ddrbin_param.txt` 文件，填入你想设置的 DDR 频率、UART 信息等参数。如果要保持默认值，可以将这些项目留空。
72|
73|使用以下命令运行工具：
74|```bash
75|./ddrbin_tool <platform> ddrbin_param.txt <output_ddr_file>
76|```
77|
78|**示例**：
79|
80|```bash
81|./ddrbin_tool px30 ddrbin_param.txt px30_ddr_333MHz_v1.13.bin
82|```
83|此示例将在 px30 平台上，根据 `ddrbin_param.txt` 中的参数生成 `px30_ddr_333MHz_v1.13.bin` 文件。
84|
85|**2.提取 `ddr.bin` 配置（功能2）**
86|
87|该功能从现有的 `ddr.bin` 文件中读取配置参数，并将这些参数导出到 `gen_param.txt` 文件中。
88|
89|**步骤**：
90|
91|使用以下命令运行工具：
92|```bash
93|./ddrbin_tool <platform> -g <output_param_file> <input_ddr_file>
94|```
95|
96|**示例**：
97|
98|```bash
99|./ddrbin_tool px30 -g gen_param.txt px30_ddr_333MHz_v1.15.bin
100|```
101|此示例会将 `px30_ddr_333MHz_v1.15.bin` 文件中的配置参数提取并导出到 `gen_param.txt` 文件。
102|
103| **注意事项**
104|
105|- 功能1和功能2是独立的，生成的 `gen_param.txt` 文件并不需要用于功能1。
106|- 详细说明和参数含义请参考工具随附的 `ddrbin_tool_user_guide.txt` 文档。
107|
108|**工具使用示例**
109|
110|```bash
111|# 修改 DDR 配置文件
112|./ddrbin_tool px30 ddrbin_param.txt px30_ddr_333MHz_v1.13.bin
113|
114|# 提取 DDR 配置参数
115|./ddrbin_tool px30 -g gen_param.txt px30_ddr_333MHz_v1.15.bin
116|```

220|整理好的脚本如下所示：
221|~~~shell
222|#!/bin/bash
223|
224|# 定义芯片型号及对应的参数文件
225|declare -A chips=(
226|    ["rk3568"]="rk3568_ddr_1560MHz_v1.23.bin"
227|    ["rk3588"]="rk3588_ddr_lp4_2112MHz_lp5_2400MHz_v1.18.bin"
228|    ["rk3562"]="rk3562_ddr_1332MHz_v1.06.bin"
229|    ["rk3576"]="rk3576_ddr_lp4_2112MHz_lp5_2736MHz_v1.05.bin"
230|)
231|
232|# 颜色代码
233|GREEN='\033[0;32m'
234|NC='\033[0m' # No Color
235|
236|# 显示选择菜单
237|echo -e "${GREEN}请选择要处理的芯片型号：${NC}"
238|echo -e "${GREEN}0: rk3568${NC}"
239|echo -e "${GREEN}1: rk3588${NC}"
240|echo -e "${GREEN}2: rk3562${NC}"
241|echo -e "${GREEN}3: rk3576${NC}"
242|echo -e "${GREEN}4: 全部处理${NC}"
243|read -p "请输入你的选择（0-4）: " choice
244|
245|# 定义选择对应的芯片名称数组
246|selected_chips=()
247|case $choice in
248|    0) selected_chips=("rk3568");;
249|    1) selected_chips=("rk3588");;
250|    2) selected_chips=("rk3562");;
251|    3) selected_chips=("rk3576");;
252|    4) selected_chips=("rk3568" "rk3588" "rk3562" "rk3576");;
253|    *) echo "无效选择！请重新运行脚本并输入有效选项。"; exit 1;;
254|esac
255|
256|# 遍历选择的芯片型号，执行相应的操作
257|for chip in "${selected_chips[@]}"; do
258|    param_file="${chip}_gen_param.txt"
259|    ddr_file="${chips[$chip]}"
260|
261|    echo "处理 $chip ..."
262|
263|    # 读取 DDR 配置到参数文件
264|    ../../tools/ddrbin_tool $chip -g $param_file $ddr_file
265|
266|    # 修改波特率
267|    sed -i 's/uart baudrate=1500000/uart baudrate=115200/' $param_file
268|
269|    # 写回修改后的参数
270|    ../../tools/ddrbin_tool $chip $param_file $ddr_file
271|
272|    # 删除参数文件
273|    rm -rf $param_file
274|
275|    echo "$chip 处理完成！"
276|done
277|
278|echo "处理完毕！"
279|
280|
281|../../tools/ddrbin_tool rk3576 -g gen_param.txt rk3576_ddr_lp4_2112MHz_lp5_2736MHz_v1.05.bin
282|~~~
