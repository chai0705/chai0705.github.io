​	我发现不再虚拟机编译确实能优化很多，至少电脑是不会卡的，这一点很重要的，我可真是个小聪明鬼，不需要看着，多执行两边就行了，以后都要多用脚本。下面这个脚本就能很好的治好我的病症，只要执行出错，就会重新去执行
~~~shell
#!/bin/bash

# 定义函数，用于执行 build.sh 程序并处理错误
execute_build() {
    ./build.sh

    # 检查 build.sh 的退出状态码
    if [ $? -ne 0 ]; then
        echo "build.sh 执行出错，继续执行..."
        execute_build
    fi
}

# 调用函数执行 build.sh
execute_build
~~~
