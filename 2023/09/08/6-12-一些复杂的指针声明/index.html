6.12 一些复杂的指针声明

12.1 与指针相关的运算符和操作

  1. 指针声明：

    • *：用于声明指针变量。例如，int* ptr;声明了一个名为ptr的指向int类型的指针变量。

  2. 地址运算符：

    • &：用于获取变量的地址。例如，int a = 10; int* ptr = &a;将变量a的地址赋给指针ptr。

  3. 间接访问运算符：

    • *：用于访问指针所指向的内存位置的值。也称为解引用操作符。例如，int a = *ptr;将指针ptr所指向的内存位置的值赋给变量a。

  4. 后置自增自减运算符：

    • ++：后置自增运算符，将指针的值增加1。
    • --：后置自减运算符，将指针的值减少1。
    • 例如，ptr++;将指针ptr的值增加1。

  5. 成员选择运算符：

    • .：用于访问结构体或联合体类型的成员。例如，struct Person { char name[20]; int age; }; Person p; p.age = 25;
    • ->：用于通过指针访问结构体或联合体类型的成员。例如，Person* ptr = &p; ptr->age = 30;

    12.2 运算符优先级：

      • 1级（从高到低）：

        • (：函数调用、表达式分组
        • .：成员选择运算符
        • ->：成员选择运算符（用于指针）
        • ++：后置自增运算符
        • --：后置自减运算符
        • +：一元正号运算符

        2级（从高到低）：

        • ++：前置自增运算符
        • --：前置自减运算符
        • *：指针解引用运算符
        • &：地址运算符

12.3 混淆定义

以下是与指针相关的一些定义的解释：

  1. *p++;：
    • 这是一个表达式，其中p是一个指针变量。
    • *p表示对指针p所指向的内存位置进行解引用，获取该位置的值。
    • p++表示对指针p进行后置自增操作，将指针向后移动到下一个内存位置。
  2. &p++;：
    • 这是一个无效的表达式。
    • &是地址运算符，用于获取变量的地址。
    • 但是，p++是一个指针的后置自增操作，返回的是指针的值而不是地址，所以不能对其应用地址运算符。
  3. &stu.a：
    • 这是一个表达式，其中stu是一个结构体变量，a是结构体stu的成员。
    • &是地址运算符，用于获取变量的地址。
    • &stu.a表示获取结构体stu中成员a的地址。
  4. int *a[10];：
    • 这是一个声明语句，定义了一个数组a，其中每个元素都是指向int类型的指针。
    • int *a[10]声明了一个包含10个元素的指针数组。
  5. int (*a)[80];：
    • 这是一个声明语句，定义了一个指针a，它指向一个包含80个int类型元素的数组。
    • int (*a)[80]声明了一个指针a，它指向一个包含80个int类型元素的数组。
  6. int *f(int);：
    • 这是一个函数声明，定义了一个名为f的函数，该函数接受一个int类型的参数，并返回一个int类型的指针。
  7. int (*f)(int);：
    • 这是一个声明语句，定义了一个指针f，它指向一个接受一个int类型参数并返回int类型的函数。
  8. *(*f)(int);：
    • 这是一个声明语句，定义了一个指针f，它指向一个函数，该函数接受一个int类型的参数，并返回一个指针。
    • *(*f)(int)声明了一个指针f，它指向一个函数，该函数接受一个int类型的参数，并返回一个指针。
  9. int *(*(*f)(int))[101];：
    • 这是一个声明语句，定义了一个指针f，它指向一个函数，该函数接受一个int类型的参数，并返回一个指向指针的指针。
    • int *(*(*f)(int))[101]声明了一个指针f，它指向一个函数，该函数接受一个int类型的参数，并返回一个指向指针的指针。
  10. int*(*T)[10];：
    • 这是一个声明语句，定义了一个指针T，它指向一个包含10个int类型指针的数组。
    • int*(*T)[10]声明了一个指针T，它指向一个包含10个int类型指针的数组。

12.4 右左法则

右左法则（Right-Left Rule）是用于解析复杂声明的方法。根据右左法则，我们从最内层的括号开始阅读声明，并向右移动，然后向左移动。每当遇到括号时，我们需要改变阅读方向。一旦解析完括号内的内容，我们跳出括号，继续解析剩余的声明，直到整个声明解析完毕。

以下是使用右左法则解析声明的示例步骤：

  1. 从最内层的括号开始：
    • ( )：未定义标识符
  2. 向右移动：
    • *：指针
    • (*f)：f是一个指针
  3. 向左移动：
    • (*f)(int)：f是一个接受一个int参数的函数
  4. 继续解析：
    • int (*f)(int)：f是一个接受一个int参数的函数，返回int

根据右左法则，我们成功解析了声明int (*f)(int)。
